Client Follow-Up Tracker

Client Follow-Up Tracker is a productivity application that helps users manage scheduled client follow-ups and send reminder emails directly from their own Gmail account.

What the app does

  • Allows users to schedule and track client follow-ups
  • Sends follow-up emails using the user’s connected Gmail account
  • Helps prevent missed or delayed client communications

Why we request Google / Gmail access

This application uses Google OAuth and Gmail API access only to send emails on behalf of the user. Emails are sent exclusively when the user initiates or schedules a follow-up action within the app.

How user data is handled

  • Email content is used only at the moment an email is sent
  • The app does not read, scan, or analyze users’ inboxes
  • Email content is not stored after sending
  • User data is not sold, shared, or used for advertising
  • User data is not used for AI or machine learning training

Users can revoke Google account access at any time from their Google Account security settings.

Transparency & compliance

Client Follow-Up Tracker complies with the Google API Services User Data Policy and requests only the minimum scopes required to provide its core functionality.
